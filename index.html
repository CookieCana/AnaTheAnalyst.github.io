Bellabeat Case Study (Python)

Google Capstone Project (Excel, Tableau, Python)

Access the project here: Bellabeat.

This project is the culmination of the Google Data Analytics Course. In it, we analyze a technology company, Bellabeat, that specializes in heralth-focused smart products. This case study concentrates on analyzing the fitness data collected by Bellabeat's smart devices to identify potential avenues for growth. Specifically, the capabilities of the Bellabeat app are explored, a key product in their lineup. The app offers users access to comprehensive health data encompassing activity levels, sleep patterns, stress levels, menstrual cycles, and mindfulness practices. By leveraging this data, users can gain a deeper understanding of their current habits and make informed choices to support their overall health and well-being.

This project highlights data cleaning, data exploration and analysis and visualization using Python libraries such as pandas, matplotlib, and numpy. The project identified major strengths and weaknesses within the company's user data by aggregating and visualizing the data through various statistical and data visualization techniques. It involved examining patterns in user behavior, identifying correlations between different variables, and drawing actionable insights to improve Bellabeat's products or services.

Movie Corrrelation Analysis

Python

Access the project here: Movie Analysis.

This project focused on analyzing a dataset of over 6000 movies. The objective was to explore correlations between different variables and gain insights into the relationships between factors such as genre, budget, ratings, and revenue.

The Python libraries used were pandas, seaborn and matplotlib and the techniques were formulating hypotheses, conducting statistical tests, and interpreting the results to draw meaningful conclusions. Additionally, exploratory data analysis techniques were then applied to examine patterns, distributions, and relationships within the data. Seaborn and matplotlib were then used to create visualizations illustrate the findings and facilitate a better understanding of the relationships between variables.

Overall, project serves as a comprehensive analysis of movie data, exploring correlations, conducting hypothesis testing, and creating visualizations to gain insights and inform decision making based on the factors influencing a movie's performance and success..

SQL and Tableau

BMI Calculator

Python

Access the project here: BMI Calculator.

This project uses Python to create a Body Mass Index (BMI) Calculator using Python.

BMI is a numerical value that provides an indication of a person's body composition and is commonly used to assess whether a person is underweight, normal weight, overweight, or obese. Although largely outdated and generally not used in modern diagnoses, this project was a fun exercise and opportunity to learn about basic programming concepts, data input and output, mathematical calculations, and how to apply these skills in developing a functional BMI calculator using Python.
